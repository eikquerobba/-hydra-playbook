ICT Concepts ที่ Hydra ใช้
DOL — Draw on Liquidity
▾
เป้าหมายที่ราคากำลังวิ่งหา

DOL ยังไม่ถึง → เทรดตามทิศไปหา (DOL = TP)
DOL ถึงแล้ว / ถูก Sweep → รอเทรดสวน (DOL ใหม่ฝั่งตรงข้าม)
IPDA — 20/40/60 Day Cycle
▾
นับย้อนหลัง 20, 40, 60 แท่ง Daily

แต่ละ Cycle สร้าง Dealing Range:
Discount Zone (ใต้ 50%) = เน้น Buy
Premium Zone (เหนือ 50%) = เน้น Sell
OTE = 62–79% ของ Range = Entry ที่ดีที่สุด
BISI / SIBI
▾
BISI (Buy Side Imbalance / Sell Side Inefficiency) = Bullish FVG → Support Zone → Entry Buy

SIBI (Sell Side Imbalance / Buy Side Inefficiency) = Bearish FVG → Resistance Zone → Entry Sell
SMT — Smart Money Technique
▾
เปิด MNQ และ MES คู่กันเสมอ

Bullish SMT: MNQ Sweep Low ใหม่ แต่ MES ไม่ → Buy Signal
Bearish SMT: MNQ Sweep High ใหม่ แต่ MES ไม่ → Sell Signal

Divergence = ฝั่งที่ไม่ Sweep คือทิศที่ราคาจะไป
MSS — Market Structure Shift
▾
สัญญาณว่า Momentum เปลี่ยนทิศ ใช้ยืนยัน Entry บน M5

Bullish MSS: ราคาทำ Lower Low แล้วกลับขึ้น Break Swing High ล่าสุด
Bearish MSS: ราคาทำ Higher High แล้วกลับลง Break Swing Low ล่าสุด

Inversion ต้องรอ MSS ก่อน Entry เสมอ
Liquidity Void
▾
ช่องว่างขนาดใหญ่ที่ราคาวิ่งแรงจนไม่มี Transaction เลย

ต่างจาก FVG ตรงที่ Void จะ Fill เต็มๆ เสมอ ถ้ามี Void ระหว่างทางไป DOL = ราคาจะผ่านไปเร็วมาก ใช้เป็น TP เพิ่มเติมได้
C → M → D (3 Phases)
▾
Consolidation → ตลาดสะสม Liquidity ทั้งสองฝั่ง เกิดนอก Macro ได้
Manipulation → Sweep Liquidity ฝั่งหนึ่ง (ล่า Stop Loss) ต้องเกิดใน Macro
Distribution/Expansion → วิ่งหา DOL ฝั่งตรงข้าม ต้องเกิดใน Macro

Hydra Entry = จุดเริ่มต้นของ Distribution หลัง Manipulation เสร็จ
